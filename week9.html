Week Nine: Input Devices

Vision

This week I set up an PIR motion sensor that could tell my computer when motion was detected and when motion ended.

Implementation

Setting up the motion sensor was very simple. The instructions and code on learning suite worked perfectly. Once practiced setting up the motion sensor I played around with how far it can detect motion (pretty dang far) and changed around one of the pins so I could potetnially combine it with my LCD.

Notes

This week went very smoothly. The only note is that the motion sensor is very sensitive and not very good at detecting continuous motion. It easily detects when a motion begins but if there is continuous motion by it, it won't acurately tell when the motion has ended.
